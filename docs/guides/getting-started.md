# Getting Started

如果你是第一次接触这个项目，建议按下面顺序阅读：

## 1. 先看 README
先理解这个项目想做什么、为什么要做、会记录哪些内容。

## 2. 再看 ROADMAP
理解当前项目处于哪个阶段，以及后续准备做什么。

## 3. 看脱敏规范
如果你想参考这里的公开记录方式，建议先看：
- `docs/desensitization-policy.md`

## 4. 看 Learning Log 和 Case Studies
这两个部分最能体现真实推进过程。

- `docs/learning-log/`
- `docs/case-studies/`

## 5. 看 Workflows
如果你更关心“这套东西到底怎么用”，再去看：
- `docs/workflows/`

## 当前状态
这个项目目前刚起步，很多内容还在逐步补全。

但这也是它的价值所在：
你能看到它不是“一次性包装出来的成品”，而是一步一步被搭起来的。
