# 2026-03-09 — 项目初始化记录

## 背景

今天决定把“王相使用和维护个人 AI 助手的过程”整理成一个公开 GitHub 项目。

目标不是单纯放几篇文章，而是把它做成：
- 学习记录
- 经验复盘
- 教程/案例展示
- 个人作品集的一部分

## 今天完成了什么

- 明确了项目定位：三者混合（记录 + 复盘 + 展示）
- 明确了公开原则：尽量细，但必须脱敏
- 明确了受众：面向所有人，新手友好
- 确定了项目名称：`WangXiang AI Journey`
- 创建 GitHub 仓库：`wangxiang-ai-journey`
- 完成 GitHub CLI 安装与登录
- 开始搭建第一版文档骨架

## 为什么这个项目值得做

很多“AI 助手项目”只展示最后结果，不展示：
- 中间踩过的坑
- 实际怎样调整工作流
- 真正遇到的麻烦
- 为什么某种方案最后被放弃

而这些内容，往往才最有参考价值。

## 当前判断

第一阶段最重要的不是堆很多内容，而是：
- 先把仓库结构搭对
- 先把公开边界定清楚
- 先让别人一进来就知道这是干什么的
