# CONTRIBUTING

感谢关注这个项目。

这个仓库当前既是公开学习记录，也是方法沉淀仓库，所以欢迎建议，但会优先保证：

- 内容真实
- 结构清晰
- 对新手友好
- 不泄露隐私与敏感信息

## 你可以怎么参与

### 1. 提建议
如果你觉得：
- 某篇内容不清楚
- 某个流程写得太跳跃
- 某段经验对新手不够友好
- 某个案例值得补充

欢迎提 Issue 说明。

### 2. 提修正
如果你发现：
- 错别字
- 死链
- 事实错误
- 术语描述不准确

也欢迎直接指出。

### 3. 提经验
如果你也在做类似的 AI 助手 / 自动化 / build in public 项目，可以分享你的实践方式。

## 当前接受的贡献类型

优先接受：
- 文档优化
- 结构建议
- 新手友好性改进
- 公开案例补充建议
- 错误修正

当前暂不优先处理：
- 方向完全不同的大改
- 与项目主线无关的功能扩展
- 会引入敏感/隐私风险的内容

## 提交建议前请注意

1. **不要提交任何密钥、token、账号密码、服务器地址等敏感信息**
2. 如果引用真实聊天或使用记录，请先脱敏
3. 尽量说明：
   - 问题是什么
   - 为什么值得改
   - 你建议怎么改

## 内容风格偏好

这个项目希望保持：
- 真实
- 清楚
- 尽量少黑话
- 对第一次接触的人也友好

如果你写的内容很专业，也尽量补充一点背景说明。

## 关于响应

当前项目还在早期阶段，响应速度不保证很快，但我会尽量：
- 及时看
- 能收的建议就收
- 不适合的会明确拒绝，而不是拖着不处理

## 安全与隐私

所有公开内容都应遵守仓库中的脱敏规范：

- [`docs/desensitization-policy.md`](docs/desensitization-policy.md)

如果你发现仓库中存在可能泄露隐私或敏感信息的内容，请优先提醒。
